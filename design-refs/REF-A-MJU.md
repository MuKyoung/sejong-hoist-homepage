## 전체 인상
정보 밀도가 높은 **딥블루 포털**. 기하학 도형(대형 사분원·원)이 콘텐츠 블록과
맞물리며 구조감을 만들고, 풀블리드 사진 타일이 페이지의 무게를 담당한다.
공공·연구기관 특유의 신뢰감 + 정돈된 격자.

## 섹션 문법
1. **헤더**: 다크 유틸바(36px: 연락처·ENG) + 메인바. 히어로 위에서는 투명(백색
   텍스트), 스크롤 시 화이트 솔리드 전환. GNB 우측 정렬, 호버 시 로열 언더라인.
2. **히어로**: 풀블리드 사진 슬라이더. 좌측 정렬 대형 헤드라인(2줄) + 상단 얇은
   라인, 좌하단 페이지네이션 `01 — 03` + 화살표. 6s 오토플레이.
   우하단에 로열 반투명 사분원 도형이 사진 위로 겹침.
3. **회사소개**: 좌측 = 로열/딥 원·사분원 조합의 기하 컴포지션(원 안에 핵심
   수치), 우측 = 타이틀 + 카피 + `+` 아이콘 행 링크(인사말/연혁/조직도).
4. **사업영역**: 3열 풀포토 타일 그리드. 하단 네이비 그라디언트 + 라벨,
   호버 시 줌 + `+` 표시.
5. **실적**: 딥블루 다크 밴드. 좌측 타이틀+카운터, 우측 화이트 케이스 카드
   가로 스크롤.
6. **뉴스/푸터**: 공지 탭 리스트 + 우측 견적 바로가기 패널. 푸터 상단에 인증
   배지 스트립(ISO·KCs), 다크 푸터.

## 세종 적용 포인트
현장 사진이 풍부한 세종에 잘 맞는 "사진 타일" 문법. 기하 도형은 크레인 훅·거더
곡률과 어울리며, 딥블루는 기존 네이비 램프의 확장으로 수용 가능.
