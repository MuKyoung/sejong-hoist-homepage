# REF-B · 연세대학교 (yonsei.ac.kr) — "모뉴멘탈 캠페인"

데모 시안: `/demo/yonsei` · 커스텀 믹서 키: `ys`

## 전체 인상
**로열블루 + 초대형 캠페인 타이포**. 히어로가 광고 캠페인처럼 거대한 숫자
("세계 42위")를 내세우고, 본문은 여백 중심 에디토리얼로 급전환한다. 부채꼴
(사분원 호) 마스크 이미지가 시그니처. 권위와 품격, 그리고 담대함.

## 토큰
| 항목 | 값 | 비고 |
|---|---|---|
| Royal | `#0E4A84` | 헤더 바·액센트 (연세 로열블루 상응) |
| Sky | `#7FB4E8 → #E8F1FA` | 히어로 그라디언트 |
| Ink | `#10243E` | 헤딩 |
| Soft | `#F4F7FB` | 뉴스 밴드 배경 |
| 타이포 | 캠페인 숫자 900/최대 200px+, 본문 헤딩은 300~400 라이트 | 대비가 핵심 |
| 마스크 | 사분원·부채꼴 (`border-radius` 단일 코너 100%) | 이미지 마스킹 |

## 섹션 문법
1. **헤더**: 항상 로열블루 솔리드 바. 백색 로고 좌측, 백색 GNB + 유틸 우측.
   투명 전환 없음 — 바 자체가 브랜드.
2. **히어로**: 스카이 그라디언트 위 **초대형 금속질 숫자**("350TON") + 위아래
   작은 캠페인 카피. 하단 중앙에 장비 사진이 부드럽게 떠오름. 웅장함의 축.
3. **회사소개**: 에디토리얼 화이트. 로열 잉글리시 슬로건 + 대형 라이트 타이포
   2줄("현장을 들어 올리는 / 세종다움") + 얇은 룰. 우측 부채꼴 마스크 사진 +
   연한 호(arc) 도형이 배경에 겹침.
4. **사업영역**: 인덱스 에디토리얼 리스트. `01 오버헤드 크레인` 행 + 얇은 룰,
   호버 시 우측에 사진 프리뷰가 페이드-스케일로 등장, 행 텍스트 인덴트.
5. **실적**: 캠페인 수치 3연속 대형 숫자(350TON · 520+ · 100%) + 라이트 라벨,
   아래 케이스 사진 스트립.
6. **뉴스/푸터**: 소프트 블루 밴드에 뉴스 카드 3장(사진 상단). 우하단 QUICK
   플로팅 dock(견적·전화·오시는길). 로열 푸터.

## 세종 적용 포인트
"350TON"이라는 강력한 한 숫자를 가진 세종에게 캠페인 타이포 문법은 최적.
부채꼴 마스크는 크레인 붐 회전 반경의 메타포로 스토리텔링 가능.
