# REF-C · 삼성중공업 (samsungshi.com) — "시네마틱 미니멀"

데모 시안: `/demo/shi` · 커스텀 믹서 키: `shi`

## 전체 인상
**풀블리드 시네마틱 영상 히어로 + 극도로 정제된 화이트 본문**. 용접 불꽃 같은
산업 현장의 스펙터클을 영상으로 보여주고, 나머지는 검정 타이포와 여백으로
절제한다. 등분할 포토 패널 스트립(제품|서비스/그린/디지털/스마트야드)이
시그니처. 대기업 중공업의 표준적 품격.

## 토큰
| 항목 | 값 | 비고 |
|---|---|---|
| Ink | `#101820` | 본문 검정 (헤딩·GNB) |
| Accent | `#1E4FA3` | 소량의 블루 포인트 (서브카피·호버) |

| 라운드 | 대형 컨테이너만 20~24px | 패널 스트립·미디어 타일 |

## 섹션 문법
1. **헤더**: 화이트 솔리드, 헤어라인 하단. 로고 좌측, 검정 13px GNB
   센터-우측, 우측 KO·햄버거 라인 2개. 최소주의.
2. **히어로**: 풀블리드 루프 **영상**(용접·크레인 구동) + 다크 스크림. 좌측
   화이트 대형 타이포가 단어 단위로 리듬 있게 등장. 하단 스크롤 인디케이터.
3. **회사소개**: 센터 섹션 헤더(타이틀 + 블루 서브카피) + 대형 원형 마스크
   사진 중앙 + 좌우 텍스트 칼럼(설립·비전·현장). 원형 = 시그니처 모티프.
4. **사업영역**: **등분할 포토 패널 스트립** — 라운드 컨테이너 안 4패널,
   각 패널 풀포토 + 백색 라벨 + 화살표. 호버 시 해당 패널이 넓어짐(flex).
5. **실적**: 센터 헤더 + 2열 대형 미디어 타일(사진 + 좌하단 백색 라벨 +
   용량 배지), 호버 줌.
6. **뉴스/푸터**: 날짜|제목|화살표 미니멀 리스트 + 다크 CTA 밴드("프로젝트를
   상담하세요") + 화이트 3열 푸터.

## 세종 적용 포인트
시공/구동 영상이 확보되면 가장 강력한 방향. 350톤 장비의 스케일을 영상
스펙터클로 직접 전달 — "웅장함" 요구에 정면으로 부합. 영상 수급 전에는
스톡 영상으로 시연.
