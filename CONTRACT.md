## 제1조 업무의 내용 (기능 범위)

### 1.1 프로젝트 개요

을은 갑의 공식 홈페이지(이하 "본 사이트")를 아래 기술 스택 기준으로 제작한다.

- **프레임워크**: Next.js 16 (App Router)
- **언어**: TypeScript 5 (strict mode)
- **스타일링**: CSS Modules
- **애니메이션**: Framer Motion 12
- **배포 환경**: Vercel (자동 CI/CD)
- **도메인**: 갑이 보유한 도메인 연결 (갑 직접 설정 또는 을이 지원)

### 1.2 페이지 구성 및 기능 범위

#### (1) 공통 레이아웃

| 항목 | 세부 내용 |
|---|---|
| 헤더 (Header) | 로고, 메인 네비게이션, 전화번호, 견적문의 CTA 버튼, 반응형 햄버거 메뉴 |
| 푸터 (Footer) | 회사정보, 링크 컬럼, 개인정보처리방침, 저작권 표시 |
| 반응형 대응 | 모바일(320px) ~ 4K(2560px) 전 해상도 |
| SEO 기본 설정 | 페이지별 title, description, Open Graph 메타태그 |
| 부드러운 스크롤 | Lenis 라이브러리 적용 |

#### (2) 홈 (메인 페이지)

| 섹션 | 세부 내용 |
|---|---|
| 히어로 (Hero) | 영상 배경 슬라이더(최대 3개 영상 자동 전환), 핵심 수치 스트립, CTA 버튼 2개 |
| 제품소개 | Swiper 슬라이더 형태의 5개 제품군 카드 (영상 썸네일, 사양, 링크) |
| 주요 실적 | 애니메이션 카운터 수치 4개 |
| 제품 라인업 | 탭 인터페이스 형태의 제품별 상세 설명 + 이미지 |
| 시공사례 | 그리드 레이아웃 카드 4개 + 주요 시공처 마퀴 롤링 |
| 최신 소식 | 공지사항/시공사례 리스트 4건 |
| 견적 문의 CTA | 영상 오버레이 배경, 연락처 카드 3개 |

#### (3) 회사소개

| 페이지 | 세부 내용 |
|---|---|
| 인사말 | 대표이사 인사말 텍스트, 이미지 |
| 오시는 길 | 지도 임베드 (카카오맵 또는 네이버맵), 주소·교통편 안내 |

#### (4) 제품소개 (5개 하위 페이지)

공통 구성: 제품 상단 히어로, 제품 설명, 사양표, 적용 사례 이미지, 문의 유도 CTA

| 페이지 | URL |
|---|---|
| Wire Hoist | /business/wire-hoist |
| Chain Hoist | /business/chain-hoist |
| Explosion-Proof Hoist | /business/explosion-proof-hoist |
| Crane | /business/crane |
| Hoist & Crane | /business/hoist-crane |

#### (5) 시공사례

- 갤러리 그리드 레이아웃 (카테고리 필터 포함)
- 개별 사례 카드: 제품명, 하중, 고객사명, 사진
- 페이지네이션 (최초 20건 표시)

#### (6) 견적문의

- 온라인 견적 문의 폼 (견적유형, 업체명, 담당자, 연락처, 이메일, 제목, 문의내용, 파일첨부)
- 개인정보 수집·이용 동의 체크박스
- 이메일 발송 연동 (EmailJS 또는 Nodemailer 기반, 갑의 이메일 수신)
- 제출 완료 안내 메시지

#### (7) 자료실

- 카탈로그·매뉴얼 PDF 파일 목록 테이블
- 파일 다운로드 기능

#### (8) 공지사항

- 리스트 뷰 + 상세 페이지
- 최초 더미 데이터 3건 포함 (향후 CMS 연동은 별도 협의)

### 1.3 범위 외 항목 (별도 협의)

다음 항목은 본 계약 범위에 포함되지 않으며, 필요 시 별도 견적을 협의한다.

- 회원가입/로그인 시스템
- 관리자 CMS(콘텐츠 관리 시스템) 구축
- 다국어(한국어 외) 지원
- 전자상거래(결제) 기능
- 외부 ERP·CRM 연동
- 검색엔진 최적화(SEO) 컨설팅 서비스
- 배너·상세 이미지 등 그래픽 디자인 신규 제작 (갑 제공 자료 활용은 포함)

### 3.1 전체 일정

- **갑의 귀책 사유**로 인한 일정 지연(자료 미제공, 검토 지연 등)은 지연 일수만큼 납품일이 자동 연장된다.
- **을의 귀책 사유**로 인한 지연은 지연 1일당 계약 금액의 0.1%를 지체상금으로 지급하되, 총 지체상금은 계약 금액의 10%를 초과하지 않는다.

## 제4조 수정 및 변경 정책

### 4.1 무상 수정 범위

계약 기간 내 다음 범위의 수정은 별도 비용 없이 처리한다.

| 구분 | 내용 |
|---|---|
| 텍스트 수정 | 문구, 연락처, 주소 등 텍스트 변경 |
| 이미지 교체 | 갑이 제공하는 동일 규격 이미지로의 교체 |
| 색상·간격 미세 조정 | 최초 확정 디자인 기준 10% 이내의 조정 |
| 오류 수정 | 을의 개발 실수로 인한 버그, 레이아웃 깨짐 |

### 4.2 유상 변경 범위

다음에 해당하는 변경은 별도 견적을 산정하여 갑의 사전 서면 승인 후 진행한다.

- 디자인 시안 확정 후 전체 레이아웃 또는 콘셉트 변경
- 제1조에 명시된 기능 범위 외 기능 추가
- 페이지 추가 (1.2항에 명시된 범위 외)
- 동일 페이지 내 3회를 초과하는 반복 수정 요청

### 4.3 수정 요청 절차

1. 갑이 이메일 또는 서면으로 수정 내용을 명확히 기재하여 을에게 요청
2. 을은 요청 접수 후 **2영업일 이내** 처리 가능 여부 및 예상 일정 회신
3. 유상 변경의 경우 을이 견적서 발송 → 갑의 서면 승인 후 작업 착수

## 제5조 납품물

### 5.1 필수 납품물

을은 최종 검수 완료 후 다음을 갑에게 납품한다.

| 납품물 | 형태 | 설명 |
|---|---|---|
| 소스 코드 전체 | GitHub 비공개 저장소 이전 또는 ZIP 압축 파일 | Next.js 프로젝트 전체 |
| 운영 서버 배포 | Vercel 운영 환경 | 도메인 연결 상태로 납품 |
| 환경 변수 파일 | `.env` 파일 (암호화 또는 직접 전달) | API 키, 이메일 설정 등 |
| 관리자 접속 정보 | 서면 또는 암호화 문서 | Vercel 계정, 도메인 관리 접속 정보 |
| 간단한 운영 가이드 | Markdown 또는 PDF 문서 | 텍스트/이미지 수정 방법, 배포 방법 |

### 5.2 납품 확인

- 을이 스테이징 URL을 갑에게 제공하면 갑은 **7영업일 이내** 검수 의견을 회신한다.
- 7영업일 내 별도 의견이 없는 경우 검수가 완료된 것으로 간주한다.
- 갑의 검수 완료 확인 이메일 또는 서면을 최종 납품 완료 기준으로 한다.

## 제6조 무상 유지보수

### 6.1 무상 유지보수 기간

납품 완료일로부터 **3개월(90일)**

### 6.2 무상 유지보수 범위

다음 사항에 한해 추가 비용 없이 지원한다.

| 유형 | 내용 |
|---|---|
| 버그 수정 | 을의 개발 결함으로 인한 기능 오류 및 레이아웃 깨짐 수정 |
| 호환성 대응 | 주요 브라우저(Chrome, Safari, Edge, Firefox) 최신 버전 기준 표시 오류 수정 |
| 배포 환경 장애 | Vercel 설정 오류 등 을의 배포 작업에 기인한 서비스 중단 복구 |

### 6.3 무상 유지보수 제외 사항

- 갑이 직접 소스 코드를 수정하여 발생한 오류
- 외부 서비스(도메인, CDN, 이메일 서비스 등) 정책 변경에 따른 영향
- 새로운 기능 추가 요청
- 자연재해, 해킹, DDoS 공격 등 불가항력에 의한 장애

### 6.4 무상 유지보수 만료 후

무상 유지보수 기간 종료 후 지속적인 유지보수가 필요한 경우, 별도 유지보수 계약을 체결한다. 월 유지보수 단가는 계약 시 협의한다.

## 제7조 장애 대응 기준

### 7.1 장애 등급 분류

| 등급 | 정의 | 예시 |
|---|---|---|
| **긴급 (P1)** | 서비스 전체 접속 불가 | 홈페이지 404·500 전면 오류, 도메인 미연결 |
| **높음 (P2)** | 핵심 기능 장애 | 견적문의 폼 미발송, 메인 영상 미재생 |
| **보통 (P3)** | 일부 기능 또는 UI 오류 | 특정 페이지 레이아웃 깨짐, 이미지 미로드 |
| **낮음 (P4)** | 사소한 UI 불일치 | 여백·폰트 크기 미세 불일치 |

### 7.2 대응 목표 시간 (무상 유지보수 기간 내)

| 등급 | 최초 응답 시간 | 해결 목표 시간 | 비고 |
|---|---|---|---|
| P1 | 2시간 이내 | 8시간 이내 (업무일 기준) | 주말·공휴일 포함 |
| P2 | 4시간 이내 | 1영업일 이내 | |
| P3 | 1영업일 이내 | 3영업일 이내 | |
| P4 | 3영업일 이내 | 5영업일 이내 | |

> 업무 시간: 평일 09:00 ~ 18:00 (공휴일 제외). P1 장애는 업무 시간 외에도 대응한다.

### 7.3 장애 접수 채널

- **이메일**: 을의 담당자 이메일 (계약서 서명 후 갑에게 별도 고지)
- **전화**: 을의 담당자 연락처 (P1 한정)
- 장애 접수 시 갑은 오류 화면 캡처 또는 URL, 발생 시각, 재현 방법을 함께 전달하면 해결 속도가 향상된다.

### 7.4 불가항력

천재지변, Vercel·AWS 등 외부 인프라 전면 장애, 사이버 공격 등 을의 통제 밖의 사유로 인한 서비스 중단은 위 대응 목표 시간의 예외로 한다. 단 을은 복구를 위해 최선의 노력을 다하며 갑에게 상황을 신속히 공유한다.

## 제8조 소유권 및 저작권

### 8.1 결과물 소유권

- 잔금 지급 완료 시점부터 본 사이트의 **소스 코드, 디자인, 콘텐츠에 대한 소유권은 갑에게 이전**된다.
- 단, 갑이 제공한 로고·영상·이미지 등 원저작물의 저작권은 갑에게 귀속된다.

### 8.2 제3자 라이선스 컴포넌트

을이 제작에 사용하는 오픈소스 라이브러리(Next.js, Framer Motion, Swiper 등)는 각각의 오픈소스 라이선스(MIT 등)를 따른다. 을은 상업적 이용이 허용된 라이선스의 구성 요소만을 사용하며, 사용 라이브러리 목록을 납품물에 포함한다.

### 8.3 을의 포트폴리오 사용

을은 본 프로젝트 결과물을 자사 포트폴리오 및 홍보 목적으로 사용할 수 있다. 단, 갑의 사전 동의 없이 소스 코드 전체를 제3자에게 공개하거나 재판매해서는 안 된다.

### 8.4 비공개 의무

을은 프로젝트 수행 과정에서 취득한 갑의 사업 계획, 고객 정보, 미공개 제품 정보 등 기밀 정보를 계약 기간 및 종료 후 3년간 외부에 공개하거나 유출해서는 안 된다.

## 제9조 보안 및 데이터 관리

### 9.1 개인정보 처리

- 본 사이트의 견적문의 폼을 통해 수집되는 개인정보(성명, 연락처, 이메일 등)는 갑의 이메일로 직접 전송되는 방식으로 처리되며, 을은 해당 정보를 별도 저장하지 않는다.
- 갑은 개인정보보호법에 따라 개인정보처리방침을 수립·공개할 의무가 있으며, 을은 사이트 내 개인정보처리방침 페이지 구현을 지원한다. 방침 내용의 법적 적합성은 갑의 책임으로 한다.

### 9.2 접근 권한 관리

- Vercel, GitHub, 도메인 관리 등 운영에 필요한 계정은 갑 명의로 생성하거나, 을이 생성 후 납품 시 갑에게 이전하는 것을 원칙으로 한다.
- 을은 프로젝트 기간 중 접근 권한을 부여받으며, 잔금 지급 및 납품 완료 후 을의 접근 권한을 해제하거나 변경비밀번호를 갑이 설정한다.

### 9.3 소스 코드 보안

- API 키, 비밀번호 등 민감 정보는 환경 변수(`.env`)로 관리하며, 소스 코드 저장소(GitHub 등)에 직접 노출되지 않도록 한다.
- 을은 알려진 취약점이 포함된 npm 패키지를 사용하지 않으며, 납품 전 `npm audit` 등으로 의존성 보안 검토를 수행한다.

### 9.4 HTTPS 적용

운영 배포 시 HTTPS(SSL/TLS)를 기본으로 적용한다. Vercel은 자동 SSL 인증서 발급을 지원하며 별도 비용이 발생하지 않는다.

## 제10조 일반 조건

### 10.1 자료 제공 의무

갑은 을이 요청하는 자료(로고, 영상, 이미지, 텍스트 콘텐츠, 회사 정보 등)를 요청일로부터 **3영업일 이내**에 제공하여야 한다. 자료 미제공으로 인한 일정 지연의 책임은 갑에게 있다.

### 10.2 담당 창구 지정

갑과 을은 각각 본 프로젝트의 담당자를 1인 이상 지정하여 계약 체결 후 상대방에게 통보한다. 의사소통은 지정된 담당자를 통해 이루어지는 것을 원칙으로 하며, 이메일로 기록을 남긴다.

### 10.3 계약 해지

- 어느 일방이 계약 조건을 중대하게 위반하고 **14일 이내** 시정하지 않을 경우 상대방은 서면 통보로 계약을 해지할 수 있다.
- **갑의 해지 시**: 을이 완료한 작업 분량에 해당하는 보수를 지급하고, 을은 해당 시점까지의 산출물을 갑에게 인도한다.
- **을의 해지 시**: 을은 기수령 착수금에서 실 투입 비용을 차감한 금액을 갑에게 환급한다.

### 10.4 손해배상

각 당사자는 자신의 귀책 사유로 상대방에게 손해를 발생시킨 경우 이를 배상한다. 단 을의 배상 한도는 본 계약 금액(VAT 제외)을 초과하지 않는다. 간접 손해(기대 이익 손실 등)에 대한 배상 책임은 부담하지 않는다.

### 10.5 분쟁 해결

본 계약과 관련한 분쟁은 먼저 협의를 통해 해결하며, 협의가 이루어지지 않을 경우 **갑의 소재지를 관할하는 법원**을 제1심 관할법원으로 한다.

### 10.6 준거법

본 계약은 대한민국 법률에 따라 해석되고 규율된다.

### 10.7 완전 합의

본 계약서는 양 당사자 간의 완전한 합의를 구성하며, 계약 체결 전 구두 또는 서면으로 이루어진 모든 협의를 대체한다. 계약 내용의 변경은 양 당사자의 서면 합의로만 가능하다.
